Installation

The library will have to be added to the project's dependencies:

SBT

{% highlight scala %} resolvers += Seq("Sonatype OSS Releases" at "https://oss.sonatype.org/content/repositories/releases") libraryDependencies += "org.talares" %% "talares" % "0.1.0" libraryDependencies += "org.talares" %% "talares-java" % "0.1.0" // Only needed to make the Java API available {% endhighlight %}

Maven

{% highlight xml %} org.talares talares_2.10 0.1.0 org.talares talares-java_2.10 0.1.0 sonatype-nexus-releases Sonatype OSS Releases https://oss.sonatype.org/content/repositories/releases {% endhighlight %}

An application.conf file will have to be present and added to the class path. This is usually done by placing it in the project's resources folder. The key talares.webservicelocation needs to be added to this file and it's value set to the URL through which the OData webservice can be reached.

Example application.conf

{% highlight haml %} talares.webservicelocation="http://www.example.com/cd_webservice/odata.svc" {% endhighlight %}

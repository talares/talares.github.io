Prerequisites

First and foremost the Tridion OData webservice needs to be installed, running and accessible via URL. The Tridion documentation can be consulted for further information on how this is done.

To make use of the library the adopting project will have to use Java 7 or above. The library is released for both Scala 2.10 and 2.11.
